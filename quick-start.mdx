## 1 – Interactive Mode

```bash
magemaker --cloud [aws|gcp|azure|all]
```

The TUI lets you:
• Deploy new models  
• List active endpoints  
• Query or delete endpoints

---

## 2 – YAML-Based Deployment (Recommended)

```bash
magemaker --deploy .magemaker_config/your-model.yaml
```

Example (AWS):
```yaml
deployment: !Deployment
  destination: aws
  endpoint_name: bert-base-demo
  instance_count: 1
  instance_type: ml.m5.xlarge

models:
  - !Model
    id: google-bert/bert-base-uncased
    source: huggingface
```

Full schema & advanced flags are documented in the <a href="/concepts/cli-reference">CLI reference</a>.

## 4 – OpenAI-Compatible Proxy
Run a local FastAPI server and plug Magemaker into any OpenAI SDK:

```bash
uvicorn server:app --reload  # default http://localhost:8000
```

```python
from openai import OpenAI
client = OpenAI(base_url="http://localhost:8000", api_key="unused")

chat = client.chat.completions.create(
    model="meta-llama/Meta-Llama-3-8B-Instruct",
    messages=[{"role": "user", "content": "Hello!"}]
)
print(chat.choices[0].message.content)
```
More on the <a href="/concepts/api-proxy">API Proxy</a> page.

---

## 5 – Clean Up & Cost Control
```bash
magemaker --delete my-endpoint
```
<Warning>Endpoints incur charges while running—delete what you don’t use!</Warning>
